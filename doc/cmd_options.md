ARGS 옵션
====

명령줄 인수를 사용하여 봇 설정을 변경할 수 있습니다.<br>
이 설정들은 json 설정값보다 우선시됩니다.

--sharding [bool]
----
샤딩을 켜고 끌 지 여부를 설정합니다.

--shard-no [int]
----
현재 인스턴스의 ID를 설정합니다.

--shard-size [int]
----
전체 인스턴스의 사이즈를 설정합니다.

커스텀 옵션
----
`ARGV` 클래스를 이용하여 스크립트에서 명령줄 인수에 접근할 수 있습니다.
```cs
string[] v = ARGV.values;
```
